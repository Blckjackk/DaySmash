Folder ini berisi dokumentasi foto komunitas DaySmash (Galeri). Tempatkan semua foto acara, sesi latihan, dan dokumentasi turnamen di folder ini.

- Struktur rekomendasi:
  - 2026-07-12_championship/ (foto turnamen)
  - 2026-05-xx_training/ (foto sesi latihan)
  - group_photos/ (foto bersama member)

Cara menggunakan:

- Gunakan nama file deskriptif dan tanggal di awal: `YYYY-MM-DD_deskripsi.jpg`.
- Untuk menambahkan ke galeri di situs, letakkan file di folder ini; komponen Galeri akan membaca file statis dari sini.

Catatan: folder ini disimpan statis, tidak ada backend. Pastikan file berukuran wajar untuk web (compressed ~200-800KB tergantung resolusi).
